Política de Privacidade

Saiba como a Lumen recolhe, utiliza e protege os dados pessoais dos utilizadores.

Última atualização: abril de 2026

1. Dados recolhidos

Podemos recolher nome, email, dados de conta, informações de compras simuladas e dados de navegação necessários ao funcionamento do site.

2. Finalidades

Os dados são usados para gerir contas, apresentar produtos, registar encomendas, melhorar a experiência e analisar visitas através do Google Analytics.

3. Google Sheets como base de dados

Este projeto utiliza Google Sheets como base de dados académica para produtos, clientes e vendas. Não deve ser usado para passwords reais ou informação sensível.

4. Google Analytics

Usamos Google Analytics para compreender visitas, páginas vistas e interação geral. Pode gerir cookies através do banner do site.

5. Direitos do utilizador

Pode solicitar acesso, correção ou eliminação dos seus dados através do email lumen@gmail.com.
